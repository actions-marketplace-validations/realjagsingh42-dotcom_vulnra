Built on NVIDIA Garak

VULNRA's open-source engine is built on NVIDIA Garak v0.14.0, the leading open-source LLM vulnerability scanner used by researchers worldwide. VULNRA extends Garak with a compliance mapping layer and LLM-as-judge scoring.

FAQ

Is VULNRA open source?
Yes, the core scan engine is MIT licensed. The web UI and PDF generator are proprietary.
Can I self-host VULNRA?
Yes, you can run the open-source CLI engine locally. The SaaS version adds enterprise features.
