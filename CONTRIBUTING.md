# Contributing to VULNRA

First off, thank you for taking the time to contribute!

All types of contributions are encouraged and valued. See the [Table of Contents](#table-of-contents) for different ways to help and details about how this project handles them.

## I Have a Question

If you have a question about how to use the scanner, please check the main documentation or open an issue using the `Question` label. 

## I Want To Contribute

### Reporting Bugs

We use GitHub issues to track public bugs. Report a bug by opening a new issue; it's that easy!

### Suggesting Enhancements

If you feel you have a valuable addition to make, such as adding a new scan module for a specific LLM vulnerability, please propose it by submitting an issue using the Feature Request or Scan Module Request template.
