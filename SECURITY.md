# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| v1.0.x  | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Please do not open a public issue. Email security@vulnra.com with:

- Description of the vulnerability
- Steps to reproduce
- Potential impact

We will respond within 48 hours.
